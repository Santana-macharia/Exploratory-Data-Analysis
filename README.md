# Steps of Data Exploration and Preparation

Data exploration, cleaning and preparation can take up to 70% of your total project time.

2.	Conda Environment
3.	Pandas
4.	Univariate Analysis
5.	Bivariate Analysis
6.	Missing Values
7.	Outliers
8.  Conclusion
